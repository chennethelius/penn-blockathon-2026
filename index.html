How It Works

From wallet to verdict in seconds.

01
Submit an address
Paste any Tron wallet into the lookup tool or call the REST API or MCP server directly from your agent.
02
Anubis scores it
The ML engine fetches 50 Tron-native features, runs XGBoost, and returns a score with Monte Carlo confidence bands.
03
Act on the verdict
Five bands from the Oracle: TRUSTED · REPUTABLE · CAUTION · RISKY · BLACKLISTED. The TrustGateContract auto-blocks untrusted wallets before any transaction is signed.
x402 Protocol

Pay-per-call via USDT-TRC20.

No API keys. No subscriptions. Send USDT on Tron, get a response. Built on the x402 HTTP payment standard.

Endpoint
GET /api/x402/trust$0.02
GET /api/x402/token-check$0.01
GET /api/x402/reputation$0.03
POST /api/x402/token-forensics$0.05
POST /api/x402/register-passport$1.00
How it works
01Call any x402 endpoint — receive a 402 with payment instructions
02Send the exact USDT amount to the treasury address on Tron
03Retry with X-Payment: <tx_hash> header — get full response
