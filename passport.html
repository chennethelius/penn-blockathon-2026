MCP Server & SDK

Built for AI agents to use

An MCP server with 5 tools so any AI agent can check trust, verify tokens, and report outcomes — no web UI needed.

mcp_config.json
{
  "mcpServers": {
    "trontrust": {
      "url": "https://app.trontrust.io/api/mcp"
    }
  }
}
guard_example.py
from trontrust_guard import GuardedWallet

# Wrap any tronpy wallet
wallet = GuardedWallet(
    private_key="...",
    min_trust_score=60
)

# Trust-gated — checks before every tx
wallet.send_usdt(
    to="TTargetWallet...",
    amount=100
)
# Raises TrustError if score < 60
MCP Tools (5)
get_agent_trust(address)
Returns trust score (0–100), verdict, risk outlook, and score breakdown for any Tron wallet.
get_token_forensics(token_address)
Deep token analysis: rug probability, honeypot detection, liquidity, freeze/mint functions.
get_agent_reputation(address)
Community reviews, sentiment analysis, average rating, and top reviewed interactions.
report_outcome(query_id, outcome, reporter)
Report a job outcome (success/failure/partial/expired). Earns 5 Sun Points and feeds Anubis retraining.
get_sun_points_balance(address)
Returns Sun Points balance, total earned, and current streak for an address.
